## Run

1. Create ns for deployments:
```bash
kubectl create ns nsm-system
```

Apply NSM resources for basic tests:
```bash
kubectl apply -k https://github.com/networkservicemesh/deployments-k8s/examples/multiforwarder?ref=a3b16f7598b24eb08c90b37bbb5452301df4dc72
```

## Cleanup

Delete ns:
```bash
kubectl delete ns nsm-system
```
